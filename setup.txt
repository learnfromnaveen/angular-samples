1. Install node js  ( LTS version )
    https://nodejs.org/en/

2.  Install angular cli  
    npm install -g @angular/cli

3. create angular app  
    ng new <app_name>  

    example: ng new my-app

4. Run the angular app  

    a. get inside the my-app directory  
    b. run the below command  
    
        ng serve --open
